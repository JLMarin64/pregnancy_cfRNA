# pregnancy_cfRNA
Repository to reproduce analyses from manuscript titled "A noninvasive blood test for fetal development predicts gestational age and preterm delivery"

# To run the code

## Install the dependencies
### R packages
* plyr
* dplyr
* ggplot2
* reshape2
* cowplot
* corrplot
* caret
* randomForest
* edgeR
* limma
* locfit
* splines
* KernSmooth
* statmod
* MASS
* exactRankTests
* pROC
* effsize

### Python packages
* seaborn
* numpy
* pandas
* matplotlib

## Download data
### Raw data
Data can be found in the folder raw_data.
